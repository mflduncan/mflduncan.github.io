Projects

Here are a few of the things I work on in my free time

Bing Bot

Uses C# and PhantomJS Webdriver to get the maximum amount of Bing Rewards points every day. Performs random searches and even spoofs user agents to get the extra mobile and Microsoft Edge searches. Supports multiple accounts, TOR functionality, and easy integration with Windows Scheduler.

 Github

Kiss Autoplay

Multibrowser extension to improve user experience on a few popular video sharing website. Adds features such as lightboxed video, ad blocking, autoplay, auto-skips (for intro and outro), and more. Grew to 1000+ users in less than 6 months and became one of the most popular extensions for the website.

 Github

Schedule Genie

Web application that generates non-overlapping WSU class schedules tailored to user and displays in a visual planner-like interface. Allows user to navigate all possible schedules easily. Uses .NET and SQLite backend with Bootstrap frontend to provide a fast, consistent user experience on all devices.

About Me

A litte more about myself

Born and raised in the Inland Northwest, I’m a big fan of the great outdoors, rock music, and almost anything technology related. I started programming my freshman year of high school and have been in love with it ever since. Curiosity has left me with a bit of experience in everything-- games, mods, desktop applications, mobile apps, and web-- but nowadays I mostly focus on web related projects. My dream is to work for a fast-moving, innovative, web company within biking distance from home.
